## 🌟 Описание

**VK Turn Proxy GUI** — это десктопное приложение, созданное для упрощения работы с [vk-turn-proxy](https://github.com/cacggghp/vk-turn-proxy). Оно избавляет от необходимости запускать прокси через командную строку, предоставляя интуитивно понятный графический интерфейс для настройки всех необходимых параметров маршрутизации и обфускации.

## ✨ Возможности

- 📥 **Управление ядром:** Скачивание и автоматическое обновление бинарных файлов ядра прямо из приложения (поддержка актуальных релизов).
- ⚙️ **Тонкая настройка:** Удобный ввод локального адреса, количества соединений, адреса пира и ссылок на ВК.
- 🔄 **Выбор режима:** Быстрое переключение между режимами работы (UDP / TCP).
- 🛠 **Продвинутые параметры:** Переопределение портов, IP-адресов TURN и возможность отключения обфускации DTLS.
- 💻 **Встроенная консоль:** Отслеживание статуса работы и логов прокси-сервера в реальном времени.
- 🔧 **Кастомные флаги:** Возможность передачи собственных параметров запуска ядру.
